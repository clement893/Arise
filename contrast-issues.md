# Problèmes de contraste identifiés

## Page Login
- Labels "Email address" et "Password" : texte sombre sur fond sombre (difficile à lire)
- Texte "Sign in or continue your leadership journey" : texte sombre sur fond sombre
- Texte "* required field" : texte sombre sur fond sombre
- Lien "Forgot password?" : texte sombre sur fond sombre
- Texte "Don't have an account?" : texte sombre sur fond sombre
- Texte "Or continue with" : texte sombre sur fond sombre
- Lien "Create account" : texte sombre sur fond sombre

## Page d'accueil (Hero)
- Les liens de navigation (Approach, Pricing, About, Sign in) semblent OK en blanc
- Le texte de description sous le titre semble légèrement sombre

## Composants à corriger
1. Input labels - doivent être blancs sur fond sombre
2. Helper text - doit être blanc/gris clair sur fond sombre
3. Links - doivent avoir un contraste suffisant
4. Dividers text - "Or continue with" doit être visible
